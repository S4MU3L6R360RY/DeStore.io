File Storage with Smart Contracts

Destore is built on Ethereum smart contracts. Smart contracts are pre-written code, which you can inspect and verify, that execute when conditions are met. A permanent, anonymous record of transactions is stored in the Ethereum blockchain, able to be referenced and verified at any time.
